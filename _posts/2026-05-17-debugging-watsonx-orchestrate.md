*Recently new ways to debug agents have been added to IBM watsonx Orchestrate. This post shows samples of the new user interface.*

There are two tools for the different types of agents:

1. Agent debugger for [agents following instructions](https://developer.watson-orchestrate.ibm.com/agents/agent_styles)
2. Flow inspector for [agentic workflows](https://developer.watson-orchestrate.ibm.com/tools/flows/overview)

## Agent Debugger

In the Orchestrate chat experience a new debug icon shows up.

![image](/assets/img/2026/05/debug-wxo-1.png)

The Agent Debugger shows the structure of the agentic application and the flow.

![image](/assets/img/2026/05/debug-wxo-2.png)

Additionally, traces can be utilized to understand agent runs.

![image](/assets/img/2026/05/debug-wxo-3.png)

## Flow Inspector

With the Flow Builder agentic workflows can be created, for example well defined steps that can utilize Large Language Models.

![image](/assets/img/2026/05/debug-wxo-5.png)

The Flow Inspector displays how workflows have been executed.

![image](/assets/img/2026/05/debug-wxo-6.png)
